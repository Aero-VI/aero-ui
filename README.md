# @aero-vi/aero-ui

AeroMail UI component library — React + TypeScript + Vite + Storybook 8 + Vitest.

## Overview

This is the shared component library powering AeroMail, Aero-VI's open-source self-hosted Gmail-clone. All shared UI components, design tokens, and utilities live here.

## Design Tokens

```ts
import { DESIGN_TOKENS } from '@aero-vi/aero-ui';
// BG_MAIN, TEXT_PRIMARY, ACCENT_BLUE, BUTTON_BLUE, FAB_TEAL, etc.
```
